# ⚛️ React Labs con TS

Percorso pratico di apprendimento React con TypeScript.
Ogni esercizio replica scenari reali utilizzati in contesto aziendale.

---

## Obiettivo

Padroneggiare i fondamentali di React con TypeScript — 
dalla gestione dello stato locale fino alla condivisione 
dello stato globale tra componenti.

---

## Contenuto attuale

### useState
Gestione dello stato locale nei componenti React.
Esercizi su contatori, liste dinamiche, stati multipli 
collegati e manipolazione di array con `filter` e `map` 
mantenendo l'immutabilità dello stato.
